How to Use

  • Task ID: Unique name for this bot session (e.g., "campaign1")
  • Access Tokens: One per line. Use multiple for rotation.
  • Thread ID: The Facebook conversation ID (numeric).
  • Messages File: Text file with one message per line.
  • Images: Select multiple images to send in rotation.
  • Interval: Time between messages (seconds) + random 5-15s jitter.
